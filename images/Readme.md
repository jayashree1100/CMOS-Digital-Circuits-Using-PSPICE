# Images for CMOS Digital Circuits Project

This directory contains all the images used in the main `README.md` of the project.

Each image serves to illustrate circuit diagrams, logic representations, and implementation visuals.

## Image Files

- **MOS.JPG**  
  Diagram of NMOS and PMOS transistors (used in the MOSFET overview section).

- **CMOS inverter.JPG**  
  CMOS inverter schematic showing NMOS pull‑down and PMOS pull‑up networks.

- **Gates.JPG**  
  Boolean expression logic gate representation used for explaining Y = (A(B + CD))'.

- **Ciruit.JPG**  
  CMOS transistor implementation diagram for the Boolean expression.
